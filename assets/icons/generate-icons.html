Icon Generator

This tool generates PNG icons in all required sizes from the SVG source file. The icons are needed for the PWA manifest and app installation.

How to use:

  1. Click "Generate All Icons" above
  2. The browser will download 8 PNG files (72x72 through 512x512)
  3. Move the downloaded files to assets/icons/ folder
  4. Refresh your app to see the new icons

Note: If automatic download doesn't work, right-click each preview size and select "Save image as..."
